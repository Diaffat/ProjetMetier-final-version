le code source contient plusieurs fichier .py

 - fichier dataexploration.py : principale fichier pour l'exploration et le pretraitement.

 - fichier preprocessing.py : contient les fonctions pour le pretraitement des données

 - fichier featureanalysis.py : contient les fonctions pour l'analyse des données

 - fichier runpredictions.py : contient des fonctions pour l'entraitenement de la la partie prediction

 - fichier St_classification : contient les modeles de machine learning qui sont prises en compte dans notre application

 - fichier plots.py : conteint les fonctions pour le dataviz,pour tracer graphes

 - le fichier df nous serve de contenir,a chaque fois qu'un dataset est uploadé et le fichier df nous serve a enregistré le dataset modifier

le dossier modelesTrain sert a enregistré les modeles sauvegardé lors de l'entraitenement
